# Nextflow

## Academy
[NextFlows](https://nextflows.ai/)

## About
This repository was created as part of the "Git & GitHub Kickoff" task 
(Week 1) for the NextFlows Academy program. It demonstrates the basics 
of setting up a GitHub account, creating a public repository, writing a 
README file, and making a first commit and push.

## How to Run
This repository currently contains documentation only (no application 
code yet). To view or use it:

1. Clone the repository:
```bash
   git clone https://github.com/DuaaNaji/Nextflow.git
```

2. Navigate into the project folder:
```bash
   cd Nextflow
```

3. Open the folder in a code editor (e.g. VS Code) to view the files.
